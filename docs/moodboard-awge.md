# Référence design — awge.com

Ajoutée au moodboard TWIY (Phase 2). Site live : https://awge.com

## Ce qu’on prend (attitude + craft)

| Pattern AWGE | Traduction TWIY |
|--------------|-----------------|
| Gate **PRESS START** (expérience, pas landing SaaS) | SoundGate = entrée de monde, pas 2 boutons plats |
| Cadre **CRT / machine** (écran = objet) | Composition “affiche / appareil”, pas page web neutre |
| **Scanlines + dither + grain** | Texture forte (proche halftone moodboard) |
| Typo **bitmap / mono commande** (`› PRESS START`) | Compteurs / microcopy mono déjà prévus — pousser le caractère |
| Mystère, peu d’explication | La marque parle d’abord ; le copy vient après |
| Site = **artefact culturel**, pas dashboard | Home publique = poster ; OS privé reste dense |

## Ce qu’on ne copie pas

- Beige CRT littéral + palette gris uniquement (TWIY reste **bleu nuit** + ponctuation rouge/jaune)
- Orange AWGE comme base (déjà écarté dans la spec)
- Pixel font partout (Syne reste display ; mono pour la donnée)
- Clone 1:1 — on prend l’**attitude**, pas le skin

## Lien avec le moodboard TWIY

AWGE renforce : gate immersif, texture lo-fi, typo agressive, anti-SaaS.  
Ça se combine avec : main+globe, chrome/vinyl SOUND, collage street, cosmos.
